#  Proyecto Integrador: Chatbot Inteligente con Docker & Git

Este proyecto consiste en el desarrollo y despliegue de un **Chatbot interactivo** basado en terminal, programado en Python. La aplicación destaca por su portabilidad, utilizando contenedores para asegurar que funcione en cualquier sistema operativo sin necesidad de instalar dependencias locales.

---

## 📋 Características Principales
* **Persistencia de Datos:** Utiliza una base de datos en formato JSON para almacenar conocimiento.
* **Contenerización:** Configuración completa mediante Dockerfile.
* **Orquestación:** Despliegue automatizado con Docker Compose.
* **Control de Versiones:** Historial de cambios gestionado profesionalmente a través de ramas en Git.

---
